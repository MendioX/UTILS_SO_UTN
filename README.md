# UTILS_SO_UTN

Utilidades en C para proyectos de Sistemas Operativos (UTN).

Incluye helpers de comunicación y funciones reutilizables
para módulos tipo Kernel / CPU / Memoria.

## Archivos
- utils.c / utils.h
- hello.c / hello.h

## Uso
Compilar junto a tu proyecto:

gcc main.c utils.c hello.c -o programa
